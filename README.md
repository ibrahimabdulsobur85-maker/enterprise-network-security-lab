# Enterprise Network Security Lab (Setup Version)

This repository provides a **step-by-step setup environment** for building a secure enterprise network using Cisco Packet Tracer.

## What You Will Build
- VLAN-based enterprise network
- Inter-VLAN routing
- DHCP configuration
- NAT for internet access
- ACL firewall rules
- Port security
- Basic cybersecurity attack simulation

## Goal
To help learners and job seekers build a **real-world enterprise network project** for networking and cybersecurity roles.

## Skills Covered
- Network design
- IP addressing & subnetting
- Cisco configuration
- Security enforcement
- Traffic analysis concepts
